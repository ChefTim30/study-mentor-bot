# Study Mentor Telegram Bot

Telegram-бот для подготовки к ЕГЭ и изучения Python.

Проект позволяет пользователю:

* зарегистрироваться и сохранить свои данные;
* выбирать направление подготовки;
* выбирать тему и уровень сложности заданий;
* получать задания из базы данных;
* проверять ответы и отслеживать прогресс.

## Используемые технологии

* Python
* python-telegram-bot
* SQLite
* Git / GitHub
* python-dotenv

## Возможности

### Регистрация пользователя

При первом запуске бот сохраняет:

* имя пользователя;
* класс обучения;
* целевой балл экзамена.

### Система заданий

Пользователь может выбрать:

1. Категорию

   * ЕГЭ
   * Python

2. Тему

3. Уровень сложности

   * Легко
   * Средне
   * Сложно

После выбора бот выдает случайное задание из базы данных.

### Проверка ответов

Бот:

* сравнивает ответ пользователя с правильным ответом;
* сообщает результат проверки;
* обновляет статистику выполненных заданий.

### Профиль пользователя

В профиле отображается:

* имя;
* класс;
* целевой балл;
* количество решённых задач.

## Структура проекта

bot.py — логика Telegram-бота

database.py — работа с базой данных SQLite

keyboards.py — клавиатуры Telegram

fill_tasks.py — заполнение базы данных задачами

.env — хранение токена бота

## Установка

1. Клонировать репозиторий

```bash
git clone <repository_url>
```

2. Создать виртуальное окружение

```bash
python -m venv venv
```

3. Активировать виртуальное окружение



```bash
venv\Scripts\activate
```

4. Установить зависимости

```bash
pip install -r requirements.txt
```

5. Создать файл .env

```env
TOKEN=YOUR_BOT_TOKEN
```

6. Запустить бота

```bash
python bot.py
```
